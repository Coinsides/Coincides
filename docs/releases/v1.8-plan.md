# Coincides v1.8 — Electron 桌面应用打包

> **主题**：将 Coincides 打包为独立桌面应用，用户双击安装即可使用  
> **前置版本**：v1.7.3（功能完整的学习系统）  
> **里程碑**：v1.x 系列最终版本，从开发阶段进入可分发阶段

---

## 一、架构设计

### Electron 主进程
- `electron/main.ts` → 启动 Express 后端 → 打开 BrowserWindow 加载 React 前端
- 数据存储在 `%APPDATA%/coincides/`（Windows）或 `~/Library/Application Support/coincides/`（Mac）
- 应用更新时数据目录不受影响

### 数据路径
| 内容 | 路径 | 更新时 |
|------|------|--------|
| SQLite 数据库 | `{userData}/data/coincides.db` | 保留 |
| 上传文件 | `{userData}/uploads/` | 保留 |
| 应用代码 | 安装目录 | 被替换 |

### 构建流程
```
npm run build:all
  ├── cd client && vite build → client/dist/
  └── tsc -p electron/tsconfig.json → dist-electron/main.js

npm run electron:dist
  └── electron-builder → release/Coincides-Setup-1.8.0.exe
```

## 三、用户安装步骤（目标体验）

1. 下载 `Coincides-Setup-1.8.0.exe`
2. 双击安装（可选择安装目录）
3. 桌面出现 Coincides 快捷方式
4. 双击打开 → 自动启动后端 + 前端 → 直接使用
5. 首次使用需在设置中填入 Anthropic API Key
