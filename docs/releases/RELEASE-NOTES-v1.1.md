# Coincides v1.1 发布说明

> 发布日期：2026-03-18

## 版本概述

UX 痛点修复 + 技术债偿还。修复了 6 个用户体验问题，建立了数据库 Migration 机制，并对核心组件进行了重构。

## ✨ 新增功能

- **数据库 Migration 机制**：自动化的数据库版本升级系统，未来版本更新不再需要删除重建数据库
- **任务系统升级**：任务新增描述、时间范围、清单（Checklist）子任务支持
- **Goal 嵌套**：目标支持父子层级关系，可以把大目标拆分为小目标
- **跨 Deck 复习选择器**：复习时可以选择多个卡组同时复习，不再局限于单个卡组

## 🛠 改进

- **日历 CRUD 补全**：日历页面可以直接创建、编辑、删除事件
- **Goal 完成率追踪**：目标页面显示整体完成进度
- **空 catch 块清理**：清理了 41 个文件中的空错误处理，提升代码健壮性
- **DeckDetail 组件拆分**：核心组件从 907 行拆分为 270 行 + 多个子组件，提升可维护性

## ℹ️ 更新方式

1. `git pull`
2. `npm run setup`
3. 重启服务
