# Coincides v1.3.1 发布说明

> 发布日期：2026-03-19

## 版本概述

补丁版本，修复了 3 个 UI bug、解决了 Windows 兼容性问题，并提升了 AI 助手的稳定性。

## 🐛 修复

- **日历事件可编辑/删除**：日历侧边栏的任务现在有可见的编辑和删除按钮（之前只能通过右键菜单操作，很难发现）
- **卡片翻转恢复正常**：修复了翻转卡片时背面内容以镜像形式透出的问题
- **公式卡片预览正常渲染**：卡片列表中的数学公式现在能正确显示，不再显示原始文本
- **新数据库启动不再崩溃**：修复了全新安装时数据库创建失败的问题

## 🛠 改进

- **Windows 兼容性**：解决了 Windows 下启动服务器时的 `ERR_UNSUPPORTED_ESM_URL_SCHEME` 错误，服务器运行时从 tsx 切换为 jiti
- **AI 助手更稳定**：
  - 复杂操作（如批量创建卡片）不再超时中断
  - 修复了对话中偶尔出现的"tool_use ids not found"错误
  - AI 记忆窗口扩大，能记住更多对话上下文

## ℹ️ 更新方式

1. `git pull`
2. `npm run setup`（有新依赖）
3. 后端启动命令变更为：`node --import jiti/register server/src/index.ts`
