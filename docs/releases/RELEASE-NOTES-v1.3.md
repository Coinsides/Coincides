# Coincides v1.3 发布说明

> 发布日期：2026-03-19

## 版本概述

引入 Time Block 时间管理系统和 AI 智能排期引擎，让学习计划真正落地到每一天的时间段。同时新增了完整的新用户入驻流程和设计宪法审计。

## ✨ 新增功能

- **Time Block 时间管理**：
  - 在日历周视图中拖拽框选创建学习时间段
  - 支持 Study / Sleep / Custom 三种类型
  - 周模板 + 单日覆盖（Override），灵活应对突发变化
  - 重叠区域自动检测并可视化提示

- **AI 智能排期引擎**：
  - AI 助手会根据你设置的 Time Block 自动分配任务到合适的日期
  - 尊重 Goal 依赖关系（先修完前置目标才排后续目标）
  - 当时间缩减时，提供中性选项让你选择如何调整，不替你做决定

- **Goal 依赖关系**：
  - 目标之间可以设置前置依赖（"先完成 A 才能开始 B"）
  - 环形依赖自动检测，不允许循环
  - Goal Manager 显示依赖标签

- **L1 入驻流程**：
  - 新用户首次打开 AI 对话时，AI 会引导你一步步设置学习计划
  - 6 步参数收集：学习目标 → 截止日期 → 时间段确认 → 颗粒度 → 特殊要求 → 生成计划
  - 每次只问一个问题，不堆积

- **Daily Brief 升级**：
  - 页面顶部显示今日学习时段概览
  - 任务旁显示"服务于哪个 Must 任务"标注

## 🛠 改进

- **设计宪法审计**：全面审查 12 个检查点，修复 3 个违规项
  - 移除了 MWF 卡片上的时间估算显示
  - 移除了循环任务的"落后进度"措辞，改为中性表述

## ℹ️ 更新方式

1. `git pull`
2. `npm run setup`
3. 重启服务
