# Coincides v1.2 发布说明

> 发布日期：2026-03-18

## 版本概述

AI 交互全面重构 + 国际化 + 新用户引导流程。AI 助手变得更聪明、更懂学习规划，同时应用支持了中英双语切换。

## ✨ 新增功能

- **新用户引导流程**：首次使用的 4 步引导——创建课程 → 上传材料 → 浏览界面 → AI 帮你规划
- **中英双语切换（i18n）**：Settings 中可切换中文 / English，全部界面文案已翻译
- **Goal Manager 升级**：完整的树形目标管理 UI，支持拖拽调整层级

## 🛠 改进

- **AI 助手重构**：
  - 角色从"学习科学顾问"重新定义为"脚手架搭建者"
  - 设计宪法三条原则硬编码为不可违反的规则
  - 新增目标拆解协议（Goal Breakdown Protocol）
  - 工具升级：支持目标拆解和优先级标注
- **日历 Course 着色**：不同课程的事件用不同颜色显示，一眼区分
- **日历 Goal 标签**：日历事件显示关联的学习目标

## 🐛 修复

- Agent 报错问题修复（A-1 遗留项）

## ℹ️ 更新方式

1. `git pull`
2. `npm run setup`
3. 重启服务
