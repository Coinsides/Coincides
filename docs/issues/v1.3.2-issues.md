## #1 — GEN 卡片白屏崩溃 (P0)

**复现步骤**：
1. 让 Agent 根据笔记生成包含 GEN 类型的卡片
2. 在 Deck Grid 中观察——GEN 卡片预览区为空
3. 点击任何 GEN 卡片——页面崩溃为纯空白

**预期行为**：GEN 卡片应正常显示预览内容，点击后应正常打开卡片详情。

## #5 — Agent 制定计划跳过目标层级 (P2)

**描述**：用户让 Agent 帮忙制定 AMATH 231 复习计划，Agent 直接在日历上创建了任务事件（MUST / RECOMMENDED 任务列表），但跳过了正确的层级流程。

**正确流程**：
1. 先创建一个目标（Goal）——如"AMATH 231 考试复习"
2. 在目标里设置阶段（Stages）——如"基础概念"、"公式推导"、"应用练习"
3. 然后才在各阶段下分配具体的日历事件/任务

**实际行为**：Agent 直接跳到第 3 步，把所有任务平铺到日历上，没有目标和阶段的结构。

**备注**：Agent 生成的学习路径内容本身设计得不错（有 Must/Recommended 分级、逻辑顺序组织），但没有按照应用的数据模型层级去组织。属于 Agent prompt 逻辑问题。
